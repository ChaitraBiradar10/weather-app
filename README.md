A simple weather app project using weather api An 
interactive Advanced Weather Application built using HTML, CSS, and JavaScript, integrated with the OpenWeatherMap API. 
This app displays real-time weather data, air quality information, and a 5-day forecast for any searched city

project structure--
------------------
Advanced-Weather-App/ │ 
├── .git/ # Git version control 
├── images/ # Weather icons / images 
├── springboot/ # Backend / future integration (optional)
├── index.html # Main HTML file
├── style.css # CSS styling file
└── README.md # Project documentation

Technologies Used
----------------------------
HTML5 – Page structure (index.html) 
CSS3 – Styling and layout (style.css)
JavaScript (ES6) – API calls & dynamic updates 
OpenWeatherMap API – Weather, forecast & air quality data

How to Run the Project
----------------------------
Download or clone the repository git clone https://github.com/your-username/advanced-weather-app.git 
Open the project folder Open index.html in any browser OR use Live Server in VS Code Enter a city name and press Enter or click
